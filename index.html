나. 프로그램과 연동되는 웹사이트, 모바일 앱, API 서비스, 클라우드 서비스에도 본 약관이 적용됩니다.

- 온라인 신청으로 간편 처리

- 환불 신청: 이메일 또는 프로그램 내 신청

- 환불 방법: 원결제 수단으로 환불

가. 일반적 환불 제한
- 청약철회 기간(7일) 경과 후
- 이용자의 책임 있는 사유로 서비스 이용 불가
- 약관 위반으로 인한 이용 정지
- 시스템 요구사항 미충족으로 인한 사용 불가
나. 부분 환불 적용
- 7일 경과 후 환불 요청: 미사용 기간 비례 환불
- 서비스 장애로 인한 이용 불가: 해당 기간 비례 환불
- 약관 변경 동의 거부: 잔여 기간 전액 환불
- 운영자 사유의 서비스 중단: 전액 환불

- 운영 종료 시 잔여 후원금 처리 방안 공지

- 예외: 운영자의 중대한 배신 행위 시

- 후원 목적과 전혀 다른 용도 사용 시
